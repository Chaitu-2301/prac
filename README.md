# Java Array Practice

This repository contains Java Array practice programs created for improving programming and problem-solving skills.

## Programs Included

- Count Odd and Even Numbers
- Find Highest Number in Array
- Find Missing Number
- Remove Duplicate Elements
- Reverse Array
- Find Second Largest Element
- Store and Display Array Elements
- Calculate Sum of Array
- Student Record Program

## Technologies Used

- Java
- Arrays
- OOP Concepts
